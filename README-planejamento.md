# Planejamento do Portfólio Fullstack

## Objetivo do projeto

Criar um portfólio moderno em Angular com tema claro/escuro, páginas públicas e área administrativa usando PO-UI/TOTVS, integrado a uma API Java Spring Boot.

## Stack frontend

- Angular
- TypeScript
- SCSS
- Angular Router
- Reactive Forms
- PO-UI na área administrativa
- HttpClient

## Stack backend

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 inicialmente
- PostgreSQL futuramente

## Páginas públicas

- Home
- Sobre
- Projetos
- Experiência
- Contato

## Área administrativa

- Login
- Dashboard
- Editar Home
- Editar Sobre
- Editar Projetos
- Editar Experiências
- Editar Contato

## MVP

- Home pública pronta
- Tema claro/escuro funcionando
- Páginas internas criadas
- Admin inicial com PO-UI
- CRUD de projetos
- API Java de projetos
- Integração Angular + Java

## Melhorias futuras

- Login real
- Upload de imagem
- Envio de e-mail
- PostgreSQL
- Deploy
